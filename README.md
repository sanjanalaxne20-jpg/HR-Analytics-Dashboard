# 📊 HR Analytics Dashboard

An interactive HR Analytics Dashboard built using to analyze employee attrition, workforce demographics, job satisfaction and other key HR metrics. This dashboard provides actionable insights to help HR teams make data-driven decisions.

---

## 📸 Dashboard Preview

![HR Analytics Dashboard](HR_Analytics_Dashboard.png)

---

## 🚀 Features

- Employee Overview
  - Total Employees
  - Active Employees
  - Attrition Count
  - Attrition Rate

- Workforce Demographics
  - Gender Distribution
  - Average Employee Age
  - Age Group Analysis

- Education Analysis
  - Education-wise Attrition
  - Education Field Filter

- Job Analysis
  - Attrition by Job Role
  - Department-wise Attrition
  - Job Satisfaction Rating

- Employee Insights
  - Marital Status Analysis
  - Interactive Filters
  - Department-wise Analysis
