# HR Analytics Dashboard

## 📌 Project Overview
This project is an interactive **HR Analytics Dashboard** built to analyze employee data, monitor attrition trends and support HR decision-making.

The dashboard provides insights into employee demographics, attrition patterns, education, department, job roles, age groups and job satisfaction.
